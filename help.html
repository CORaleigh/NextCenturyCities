Map Navigation

  • Pan: Left-click and hold down left mouse button, then move mouse in desired direction
  • Tilt & Rotation: Right-click and hold down right mouse button to tilt and rotate the map
  • Zoom: Use mouse wheel button to zoom in and out

Area Plan Info

  1. Click Area Plan Info tool tab
  2. Click information panel to see area planning information
  3. Click Area Plan Info tool tab again to close the tool

Explorer Scenario Tool

The scenario tool allows users to explore alternative planning scenarios by altering existing buildings and creating new buildings

User can:

  • modify dimensions (width, depth, and rotation)
  • modify the number of floors for retail, office and/or residential use
  • create new buildings
  • see summary reporting of the selected building modification(s)
  • see total gross square foot change of the scenario at the area planning level

Cast Shadow Tool

  1. Click Cast Shadow tool tab. A clock will appear at the left bottom of the screen
  2. Rotate the hands of the clock to show shadow casting at any given time (the day changes at midnight).
  3. Revert to current time by clicking the Live button. The Live button turns red when the clock is set to the current time.
  4. Click Cast Shadow tool tab again to hide the tool and shadow

Layers Tool

  1. Click Layers tool tab
  2. Select layers to turn on and off
  3. Click Layers tool tab to close the tool

Basemap Tool

  1. Click Basemap tool tab
  2. Select a basemap to turn on and off
  3. Click Basemap tool tab to close the tool
